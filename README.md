# FitFunnel HTML/CSS Template

A clean static version built with only:
- HTML
- CSS

## Files
- `index.html` -> main page
- `styles.css` -> all styles, grouped and commented
- `assets/` -> put images, logos, videos here later

## Edit quickly
- Change text directly in `index.html`
- Replace placeholders like `Coach Image` and `Workout Reel / Intro Video`
- Add your real images inside `assets/`
- Update links, email, phone, and social handles in the footer

## Open locally
Double-click `index.html`
or open it in your browser.

## Best next step
If you want, this can later be split into:
- `index.html`
- `about.html`
- `programs.html`
- `contact.html`

for even easier editing.
